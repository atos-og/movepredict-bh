# Backend

Backend do projeto feito com Python e FastAPI.

## Como rodar

Ative o ambiente virtual:

```powershell
.venv\Scripts\activate
```

Instale as dependências:

```powershell
pip install -r requirements.txt
```

Rode o servidor:

```powershell
uvicorn app.main:app --reload
```

Acesse no navegador:

```txt
http://127.0.0.1:8000
```

Documentação da API:

```txt
http://127.0.0.1:8000/docs
```
