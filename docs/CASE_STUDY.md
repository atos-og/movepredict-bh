# Case Study — MovePredict BH

Este documento será preenchido ao longo do projeto.

## Problema

Usuários do transporte público precisam de informações mais claras sobre localização, atraso e previsão de chegada dos ônibus.

## Solução

Uma plataforma web que usa dados públicos para mostrar ônibus em tempo real, estimar chegada e gerar insights sobre confiabilidade das linhas.
